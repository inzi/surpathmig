# Organization Unit Services

## Overview
Application services for organizational unit (OU) management including hierarchical organization structure, user assignment, and role/permission inheritance.

## Key Features
- OU CRUD operations
- Hierarchical tree management
- Move OUs in hierarchy
- Assign users to OUs
- OU-based role assignment
- OU-based permission filtering

## Business Logic
Manages organizational hierarchy (departments, divisions, teams), handles user-OU assignments, enables OU-based data filtering and authorization, and supports role inheritance through OU tree.

## Usage
Consumed by organizational structure management UI and data filtering based on user's OU membership.
