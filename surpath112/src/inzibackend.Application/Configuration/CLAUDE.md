# Configuration Services

## Overview
Application services for managing host and tenant configuration settings, providing administrative interfaces for system customization.

## Subfolders
- **Host**: System-wide configuration for platform administrators
- **Tenants**: Tenant-specific configuration settings

## Key Features
- Email settings (SMTP)
- Security settings (password policies)
- UI customization (themes, logos)
- Feature toggles
- Notification preferences
- External service integration settings

## Usage
Consumed by settings UI pages for host and tenant administrators.
