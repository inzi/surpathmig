# Host Configuration Service Documentation

## Overview
Application service managing host-level (platform-wide) configuration settings. Controls system-wide settings that apply across all tenants.

## Key Features
- Email settings (SMTP configuration)
- Security settings (password policies, lockout rules)
- Feature configuration
- UI customization settings
- External service integrations
- Default tenant settings

## Usage
Provides host admin interface for system configuration. Settings stored in database and cached for performance.
