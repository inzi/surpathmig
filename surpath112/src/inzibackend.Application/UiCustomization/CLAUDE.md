# UI Customization Services

## Overview
Application services for UI theme and customization including theme selection, custom CSS, and UI preference management.

## Key Features
- Get available themes
- Change user theme
- Custom CSS management
- UI layout preferences
- Menu customization

## Business Logic
Manages UI theme and appearance settings per user, provides theme data for CSS loading, and persists user UI preferences.

## Usage
Consumed by theme selector, UI preference pages, and application layout components.
