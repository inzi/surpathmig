# Multi-Tenancy Services

## Overview
Application services for tenant management including tenant CRUD, subscription management, and tenant feature configuration.

## Contents
- Tenant creation and registration
- Tenant editing and configuration
- Tenant activation/deactivation
- Subscription management
- Edition assignment
- Feature availability per tenant

## Subfolders
- **Accounting**: Invoice and payment accounting
- **HostDashboard**: Platform-wide metrics for host admins
- **Payments**: Multi-gateway payment processing

## Key Features
- Tenant lifecycle management
- Subscription and billing
- Feature toggling per tenant
- Tenant statistics and reporting

## Usage
Core multi-tenancy infrastructure consumed by tenant registration, subscription management, and host administration interfaces.
