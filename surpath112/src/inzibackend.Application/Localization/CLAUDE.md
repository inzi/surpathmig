# Localization Services

## Overview
Application services for multi-language support including language management, translation editing, and localization resource management.

## Key Features
- Get available languages
- Change application language
- Get localized texts
- Edit translations (if enabled)
- Export/import language resources

## Business Logic
Manages application languages and translations, provides runtime language switching, and supports translation management for multi-language deployments.

## Usage
Consumed by language selector UI, localization middleware, and translation management interfaces.
