# Cache Management Services

## Overview
Application services for cache management including cache inspection, clearing, and performance monitoring.

## Key Features
- List all cache items
- Clear specific cache
- Clear all caches
- Cache statistics and metrics
- Cache configuration

## Business Logic
Provides administrative interface to ABP caching system, allows clearing caches after configuration changes or data updates, and provides visibility into cache utilization.

## Usage
Consumed by admin cache management UI and maintenance/troubleshooting tools.
