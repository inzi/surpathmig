# Logging Services

## Overview
Application services for log viewing and management, providing access to application logs for troubleshooting and monitoring.

## Key Features
- Query application logs
- Filter by level, logger, date
- View log details
- Download logs
- Clear old logs

## Business Logic
Provides administrative interface to application logging system, enables log querying and filtering for troubleshooting, and supports log retention management.

## Usage
Consumed by log viewer UI and diagnostic/troubleshooting tools.
