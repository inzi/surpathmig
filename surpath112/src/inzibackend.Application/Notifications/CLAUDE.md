# Notification Services

## Overview
Application services for user notification management including notification delivery, notification history, and notification preferences.

## Key Features
- Send notifications (in-app, email, SMS)
- Get user notifications
- Mark as read/unread
- Delete notifications
- Notification preferences management
- Notification subscriptions

## Business Logic
Manages notification lifecycle from creation to delivery, tracks read status, respects user preferences for notification channels, and provides notification history.

## Usage
Consumed by notification UI components, background jobs sending notifications, and user preference management.
