# Dynamic Entity Properties Services

## Overview
Application services for dynamic property system allowing runtime addition of custom fields to entities without code changes.

## Key Features
- Define custom properties for any entity
- Support multiple data types (string, number, date, boolean)
- Property validation rules
- Entity-specific property values
- Property groups and categories

## Business Logic
Enables extensibility by allowing administrators to add custom fields to entities dynamically, supporting tenant-specific customization requirements without code deployment.

## Usage
Consumed by entity management UIs and property value management interfaces.
