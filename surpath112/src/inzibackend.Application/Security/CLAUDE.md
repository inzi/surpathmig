# Security Services

## Overview
Application security services including password complexity validation, two-factor authentication management, and reCAPTCHA validation.

## Subfolders
- **Recaptcha**: Google reCAPTCHA validation services

## Key Features
- Password complexity checking
- Two-factor authentication setup
- Security token generation/validation
- Account lockout management
- Security log monitoring

## Usage
Consumed by authentication flows, registration, password change, and security settings management.
