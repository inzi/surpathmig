# Edition Management Services

## Overview
Application services for managing subscription editions (plans/tiers) in the SaaS platform, including feature sets, pricing, and edition lifecycle management.

## Key Features
- Edition CRUD operations
- Feature assignment to editions
- Pricing configuration per edition
- Trial edition support
- Edition comparison data
- Tenant edition assignment

## Business Logic
Defines subscription tiers with different feature sets and pricing, manages edition lifecycle, provides edition comparison for tenant upgrade decisions, and integrates with payment and tenant management systems.

## Usage
Consumed by subscription management, tenant registration, upgrade wizards, and pricing pages.
