# Excel Data Exporting Infrastructure

## Overview
Base infrastructure for Excel export functionality across the application. Contains NPOI and EPPlus base classes for consistent Excel generation.

## Subfolders
- **NPOI**: Active NPOI-based exporters (open-source)
- **EpPlus**: Disabled EPPlus-based exporters (licensing issues)

## Usage
All Excel export services inherit from base classes in these folders to ensure consistent formatting and functionality.
