# Friendship Management Services

## Overview
Application services for user friendship/connection system enabling social features like friend requests, acceptance, blocking, and friend lists.

## Key Features
- Send friend requests
- Accept/reject friend requests
- Block/unblock users
- View friend list
- View pending requests
- Friend search

## Business Logic
Manages bidirectional user relationships, handles friend request workflows, enforces blocking rules (blocked users can't message), and provides friend list data for chat and social features.

## Usage
Consumed by chat system, social features, user search, and connection management UI.
