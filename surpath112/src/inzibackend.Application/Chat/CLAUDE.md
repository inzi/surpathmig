# Chat Services

## Overview
Real-time chat application services for user-to-user messaging, including message history, user blocking, and message management.

## Contents
- Message sending and receiving
- Chat history queries
- User blocking/unblocking
- Message read status tracking
- Online user presence

## Subfolders
- **Exporting**: Chat history export to Excel

## Key Features
- Real-time messaging via SignalR
- Message persistence and history
- Read receipts
- User blocking
- Online status indicators

## Usage
Consumed by chat UI components and SignalR hubs for real-time communication.
