# Permission Management Services

## Overview
Application services for permission management including dynamic permission definition, permission tree generation, and permission assignment.

## Key Features
- Get all permissions as hierarchical tree
- Get permissions for role
- Get permissions for user
- Update role permissions
- Update user permissions
- Reset to default permissions

## Business Logic
Provides permission data structures for authorization UI, manages permission assignments at role and user levels, and handles permission inheritance.

## Usage
Consumed by permission management UI and authorization attribute validation.
