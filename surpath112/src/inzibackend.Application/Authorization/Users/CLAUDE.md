# User Management Services

## Overview
Comprehensive user management application services including CRUD operations, profile management, delegation, importing, exporting, and password management.

## Contents
- User CRUD operations
- User list queries with filtering
- Profile management services
- User activation/deactivation
- Role assignment
- Permission management
- Organizational unit assignment

## Subfolders
- **Profile**: User profile and profile picture management, SMS verification caching
- **Delegation**: User delegation for vacation/coverage
- **Importing**: Bulk user import from Excel
- **Exporting**: User data export to Excel
- **Password**: Password expiration background worker

## Key Features
- Advanced filtering and searching
- Excel import/export
- Profile management with image upload
- User delegation system
- Password expiration enforcement

## Usage
Core services consumed by user management UI, admin interfaces, and authentication systems.
