# Authorization Services

## Overview
Complete authorization and access control services including user management, role management, permission management, and account operations.

## Subfolders
- **Users**: User CRUD, profile, delegation, import/export, password management
- **Roles**: Role management and permission assignment
- **Permissions**: Permission tree and assignment services
- **Accounts**: Registration, login assistance, password reset

## Key Features
- User lifecycle management
- Role-based access control
- Permission management
- Account registration and activation
- User import/export
- Delegation system

## Usage
Core authorization infrastructure consumed throughout the application for access control and user management.
