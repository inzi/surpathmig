# Timing Services

## Overview
Application services for timezone and time-related operations including timezone information, time conversion, and user timezone management.

## Key Features
- Get available timezones
- Convert times between timezones
- Get user timezone
- Update user timezone preference
- Server time information

## Business Logic
Provides timezone data for user selection, handles time conversion for display and storage, and manages user timezone preferences for proper time display.

## Usage
Consumed by user profile, time display components, and date/time pickers throughout the application.
