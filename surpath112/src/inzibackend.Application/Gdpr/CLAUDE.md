# GDPR Compliance Services

## Overview
Application services supporting GDPR (General Data Protection Regulation) compliance including data export, data deletion, and privacy management.

## Key Features
- Export user data (right to data portability)
- Delete user account and data (right to erasure/forget)
- Data processing consent management
- Privacy policy acceptance tracking
- Data retention policies

## Business Logic
Implements GDPR requirements for data subject rights, collects all user data across system for export, performs complete data deletion with audit trail, and manages consent tracking.

## Usage
Consumed by user profile privacy settings, GDPR request processing workflows, and compliance reporting.
