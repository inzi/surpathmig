# Dashboard Customization Services

## Overview
Application services for customizable dashboard system allowing users to create, arrange, and personalize dashboard pages and widgets.

## Contents
- Dashboard page management
- Widget configuration
- Layout persistence
- User preferences
- Dashboard templates

## Subfolders
- **Dto**: DTOs for dashboard definitions and widgets

## Key Features
- Multiple dashboard pages per user
- Drag-and-drop widget arrangement
- Widget show/hide toggles
- Dashboard sharing (optional)
- Reset to defaults

## Usage
Consumed by dashboard UI for loading and saving user dashboard configurations.
