# Audit Logging Services

## Overview
Application services for querying and managing audit logs, tracking all system activities for security and compliance purposes.

## Contents
- Audit log query with advanced filtering
- Export to Excel functionality (Exporting subfolder)
- Entity change tracking
- Action auditing

## Subfolders
- **Exporting**: Excel export for audit logs

## Key Features
- Query audit logs by date, user, action, entity
- View entity changes (old/new values)
- Export audit data for compliance
- Long-term audit retention

## Usage
Consumed by audit log viewer UI, compliance reporting, and security investigation tools.
