# Audit Log Export Service Documentation

## Overview
Excel export functionality for audit logs. Provides data export of system activity for compliance auditing and security analysis.

## Contents
Excel exporter for audit log entries with filtering by date, user, action, and entity.

## Key Features
- Export audit logs to Excel
- Filter by date range, user, or entity type
- Include entity changes and old/new values
- Timezone-aware timestamps

## Usage
Used by administrators and auditors to export system activity logs for compliance reviews and security investigations.
