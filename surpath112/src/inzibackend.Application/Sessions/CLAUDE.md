# Session Services

## Overview
Application services for user session management including session information retrieval and multi-session management.

## Key Features
- Get current session info
- List user's active sessions
- Terminate sessions
- Session timeout management
- Concurrent session limits

## Business Logic
Provides current user and tenant context, manages session lifecycle, enables session termination for security, and tracks active user sessions.

## Usage
Consumed by authentication middleware, user profile, session management UI, and security monitoring.
