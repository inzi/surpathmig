## Summary
The RecordNotesExcelExporter class extends NpoiExcelExporterBase to create an Excel file containing record notes. It includes headers for various fields such as Note, Created, AuthorisedOnly, HostOnly, SendNotification, RecordState, User Name, and another User Name2. The exporter converts the Created date using a timezone converter and formats it with 'yyyy-mm-dd'. Each row's third column is formatted accordingly.
