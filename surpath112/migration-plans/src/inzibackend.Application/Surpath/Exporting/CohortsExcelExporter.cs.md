## Summary
CohortsExcelExporter is a class that extends NpoiExcelExporterBase and implements ICohortsExcelExporter interface. It is responsible for exporting cohort data to an Excel file named Cohorts.xlsx. The class constructor initializes timezone converter and ABP session, while the ExportToFile method creates and populates the Excel sheet with cohort information.
