## Summary
A custom medical units Excel exporter class that extends NpoiExcelExporterBase and implements IMedicalUnitsExcelExporter interface. It exports a list of GetMedicalUnitForViewDto objects into an Excel file named 'MedicalUnits.xlsx' with headers including Name, PrimaryContact, PrimaryContactPhone, PrimaryContactEmail, Address1, Address2, City, State, ZipCode, and HospitalName.
